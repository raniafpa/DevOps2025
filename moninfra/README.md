Présentation

Ce projet Ansible automatise le déploiement complet d'une stack Web + Base de données sur deux machines distinctes :

Serveur Web (webservers)

Installation Apache + PHP

Déploiement d’un site via Git

Configuration des permissions

Pare-feu UFW adapté

Serveur Base de données (dbservers)

Installation MySQL/MariaDB

Création base + utilisateur

Ouverture des ports nécessaires

UFW sécurité DB

Les mots de passe sensibles sont stockés et chiffrés via Ansible Vault
